Web安全

云主机一台（腾讯云、阿里云、海外云都可以）

我用Windows SERVER 2016 系统图形化方便管理
突然想到搭在云上有风险！！！
本地虚拟机同样方法

需要安装的软件

1.phpStudy （因为web漏洞靶场大多数都是基于PHP+Mysql的）

2.火狐和谷歌浏览器（方便调试）

3.Sublime Text3

4.Notepad++

5.好压（解压缩软件）

6.Navicat（数据库管理工具）
