Linux 源码安装Apache

2018-12-04

Linux

CentOS 7 源码安装 Apache httpd-2.4.37

官方安装手册http://httpd.apache.org/docs/2.4/install.html#customize

一、首先准备安装开发者工具包

  • Development Tools 安装命令：yum groupinstall "Development Tools" -y

二、下载安装Apache所需的源码包

进入/usr/loacl/src目录下载

  • apr-1.6.5.tar.gz
    wget https://mirrors.tuna.tsinghua.edu.cn/apache//apr/apr-1.6.5.tar.gz
  • apr-util-1.6.1.tar.gz
    wget https://mirrors.tuna.tsinghua.edu.cn/apache//apr/apr-util-1.6.1.tar.gz
  • pcre-8.42.tar.gz
    wget ftp://ftp.csx.cam.ac.uk/pub/software/programming/pcre/pcre-8.42.tar.gz
  • httpd-2.4.37.tar.gz
    wget https://mirrors.tuna.tsinghua.edu.cn/apache/httpd/httpd-2.4.37.tar.gz

三、编译安装下载好的源码包

在当前目录/usr/local/src下解压下载好的源码包

  • tar -xzvf apr-1.6.5.tar.gz
  • tar -xzvf apr-util-1.6.1.tar.gz
  • tar -xzvf pcre-8.42.tar.gz
  • tar -xzvf httpd-2.4.37.tar.gz

#cd /usr/local/src
#tar -xzvf apr-1.6.5.tar.gz
#tar -xzvf apr-util-1.6.1.tar.gz
#tar -xzvf pcre-8.42.tar.gz
#tar -xzvf httpd-2.4.37.tar.gz

安装apr-1.6.5

  1. cd /usr/local/src/
  2. cp -r apr-1.6.5 /usr/local/src/httpd-2.4.37/srclib/apr
  3. cd apr-1.6.5
  4. ./configure --prefix=/usr/local/apr
  5. make && make install

#cd /usr/local/src/
#cp -r apr-1.6.5 /usr/local/src/httpd-2.4.37/srclib/apr
#cd apr-1.6.5
#./configure –prefix=/usr/local/apr
#make && make install

安装apr-util-1.6.1

  1. cd /usr/local/src/httpd-2.4.37
  2. cd /usr/local/src/apr-util-1.6.1
  3. ./configure --prefix=/usr/local/apr-util --with-apr=/usr/local/apr
  4. make && make install

#cd /usr/local/src/httpd-2.4.37
#cd /usr/local/src/apr-util-1.6.1
#./configure –prefix=/usr/local/apr-util –with-apr=/usr/local/apr
#make && make install

执行第3,4步出现报错

#./configure –prefix=/usr/local/apr-util –with-apr=/usr/local/apr
#make && make install

报错信息：
解决办法：

问题1：checking for APR… configure: error: the –with-apr parameter is incorrect.
解决：
参考链接

======= 编译安装apr –util ========

mkdir /usr/local/apr/util

tar zxvf apr-util-1.3.12.tar.gz

cd apr-util-1.3.12

./configure –prefix=/usr/local/apr/util/apr-util-1.3.12 –with-apr=/usr/local/apr/apr-1.4.5

问题2：xml/apr_xml.c:35:19: 致命错误：expat.h：没有那个文件或目录 #include <expat.h>

解决：
缺少expat-devel包
yum install expat-devel

安装expat-devel包
重新进行编译安装就可以了

安装pcre-8.42

  1. cd /usr/local/src/pcre-8.42
  2. ./configure
  3. make && make install

#cd /usr/local/src/pcre-8.42
#./configure
#make && make install

安装httpd-2.4.37

  1. cd /usr/local/src/httpd-2.4.37
  2. ./configure --with-included-apr --prefix=/usr/local/apache2.4 --with-apr=/usr/local/apr --with-apr-util=/usr/local/apr-util --enable-so --enable-mods-shared=most
  3. make && make install

#cd /usr/local/src/httpd-2.4.37
#./configure –with-included-apr –prefix=/usr/local/apache2.4 –with-apr=/usr/local/apr
–with-apr-util=/usr/local/apr-util –enable-so –enable-mods-shared=most
#make && make install

安装部分到此完成

四、编写启动脚本

vim httpd
#!/bin/bash
# chkconfig: 12345 80 90
function start_http()
{
/usr/local/apache2.4/bin/apachectl start
}
function stop_http()
{
/usr/local/apache2.4/bin/apachectl stop
}
case “$1” in
start)
start_http
;;
stop)
stop_http
;;
restart)
stop_http
start_http
;;
*)
echo “Usage : start | stop | restart”
;;
esac

五、加入系统服务

chmod a+x httpd
cp -arf httpd /etc/init.d/

六、启动httpd服务

systemctl daemon-reload
systemctl start httpd

七、设置开机自启

chkconfig –add httpd

八、关闭CentOS防火墙

systemctl stop firewalld.service #停止防火墙服务
systemctl disable firewalld.service #禁用防火墙开机启动服务

九、访问httpd服务

访问Centos的httpd服务
http://192.168.17.200:80
It works!
httpd配置文件路径：
/usr/local/apache2.4/conf/httpd.conf
